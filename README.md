# 📊 E-Commerce Sales Performance Dashboard

This project focuses on analyzing and visualizing e-commerce sales data using **Power BI**.  
The dashboard provides clear insights into sales performance, customer segments, and regional trends to support better business decisions.

---

## 🎯 Project Objectives

The goal of this project is to:
- Understand overall sales and profit performance  
- Identify top-performing product categories and customer segments  
- Analyze regional sales distribution  
- Present insights in an easy-to-understand visual dashboard  

---


---

## 📈 Dashboard Highlights

- Sales and profit trends over time  
- Category-wise and segment-wise sales comparison  
- Regional sales analysis using visual charts  
- Key performance indicators (KPIs) for quick insights  

---

## 🔍 Key Insights

- **Technology** category generates the highest sales  
- **Consumer** segment dominates overall revenue  
- **West region** contributes the most to total sales  
- Profits vary significantly across regions and categories  

---

## 🛠 Tools & Technologies Used

- **Power BI** – Data visualization and dashboard creation  
- **Excel / CSV** – Data source  
- **GitHub** – Version control and project hosting

## 🚀 How to Use This Project

1. Download the `E-COMMERCE SALES PERFORMANCE DASHBOARD.pbix` file

3. Explore the interactive visuals and filters
